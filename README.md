
# PILEV Lopesan Phonegap

App que muestra los valores  producidos por cada hotel. Aplicacion diseñada para la ULPGC

PILEV: Punto de Información Lopesan de las Energías Verdes. En él se muestra la cantidad de energía solar que cada uno de los hoteles del grupo LOPESAN genera en cada minuto, así como sus parámetros equivalentes que indican el grado de protección medioambiental aplicada.
